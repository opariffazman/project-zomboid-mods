title=Item Tags Compatibility Patch
description=[b][h1]ITCP - Item Tags Compatibility Patch[/h1][/b]
description=[b]Compatibility patches for various other mods that uses hardcoded item name, this mod patches them to use in-game existing tags instead.[/b]
description=
description=[h2]Problem[/h2]
description=[list]
description=[*]Items added by [url=https://steamcommunity.com/workshop/filedetails/?id=2870742794]Tools of The Trade[/url] correctly uses in-game [b]Tags[/b].
description=[*]The Spiffo Army Knife has [i]"Scissors"[/i] and [i]"SewingNeedle"[/i] [b]Tags[/b].
description=[*]But [b]actions[/b] from mods [url=https://steamcommunity.com/workshop/filedetails/?id=2584991527]Auto Sewing[/url] won't work with it.
description=[*]As it will only works if you specifically have these items [i]"Scissors"[/i] & [i]"Needle"[/i].
description=[*]So tools from other mods that are tagged correctly won't be usable with other mods (pre tags patch) that have hardcoded values.
description=[/list]
description=
description=[h2]Solution[/h2]
description=[list]
description=[*]This mod fixes that by using the corresponding [b]Tags[/b] instead.
description=[*][strike]Recipes or[/strike] Only [b]Actions[/b] added by these patched mods will correctly recognize the items added.
description=[*]Optional but highly recommended: [url=https://steamcommunity.com/workshop/filedetails/?id=2942762545]Universal Recipes[/url] mod that fix all [b]recipes[/b] (pre tags patch) in more efficient way.
description=[/list]
description=
description=[h2]Installation[/h2]
description=Subscribe to this mod normally and enable it.
description=ITCP will automatically run patch if any of the patched mod is installed.
description=
description=[h2]Patch lists:[/h2]

description=[*][url=https://steamcommunity.com/workshop/filedetails/?id=2875848298]Common Sense[/url] - [i]Crowbar[/i] tag to Pry Doors
description=[*][url=https://steamcommunity.com/workshop/filedetails/?id=2923979345]Crafting Enhanced Core[/url] - [i]Screwdriver & Hammer[/i] tags to Create Crafting Tables
description=[*][url=https://steamcommunity.com/workshop/filedetails/?id=2803794394]For Science![/url] - [i]Scissors & SharpKnife[/i] tags to Dissect Corpse & Tinker
description=[*][url=https://steamcommunity.com/workshop/filedetails/?id=2012990668]Reorganized Info Screen[/url] - [i]Razor[/i] tag to Cut Hair/Beard etc

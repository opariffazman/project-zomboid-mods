title=ItemTagsCompatibilityPatch
description=Compatibility patches for mods that uses Base Item to use vanilla Item Tags instead.
description=
description=For example, items added by [url=https://steamcommunity.com/sharedfiles/filedetails/?id=2870742794]Tools of The Trade[/url] correctly use [b]Tags[/b] but other mods that uses Base Item instead of Tags won't recognize it.
description=
description=Mainly used for our private server with other list of mods.
description=
description=This mod fixes that by allowing other mods specific actions to be done as long as an Item with matching [b]Tags[/b] are defined in any other mods using [b]Tags[/b].
description=
description=Patch included:-
